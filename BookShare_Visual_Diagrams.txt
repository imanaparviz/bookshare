BEZIEHUNGEN:
═══════════
• User ➔ Books (1:N) - Ein User besitzt viele Bücher
• User ➔ Loans (1:N) - Ein User kann viele Bücher ausleihen/verleihen  
• Book ➔ Loans (1:N) - Ein Buch kann mehrere Ausleihvorgänge haben
• Loan ➔ Conversation (1:1) - Jede Ausleihe hat eine Unterhaltung
• Conversation ➔ Messages (1:N) - Eine Unterhaltung hat viele Nachrichten
• User ➔ Ratings (1:N) - Ein User kann viele Bewertungen abgeben
• Book ➔ Ratings (1:N) - Ein Buch kann viele Bewertungen erhalten
